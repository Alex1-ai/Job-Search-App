# Expo Router Example

Use [`expo-router`](https://docs.expo.dev/router/introduction/) to build native navigation using files in the `app/` directory.

## 🚀 How to start the project

```sh
git clone https://github.com/Alex1-ai/Job-Search-App

cd Job-Search-App

npm install

npm start

```
